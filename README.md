## Framework Design
[Spring 3.9]
- Annotation Driven을 통한 컨트롤러 호출
- Resource Mappting을 통한 리소스 폴더 관리

[MyBatis]
- MyBatis와 Oracle 연동 구현
- Connection Pool 셋팅을 통한 DB 연결 최적화
